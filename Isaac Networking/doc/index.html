    Isaac Networking System Setup

    Usage guide.

    Client Setup, Step by Step

    1. Subscribe to IsaacNetwork on Steam.
    2. Add --luadebug in Steam > Library > The Binding of Isaac: Rebirth > Properties > Set Launch Options. (For security reasons, remove it when not using the client)
    3. Subscribe to the plugins you want to use.
    4. If your server does not use the default port 21666, please modify isaacnetwork_somenumber/main.lua, local currentPort = 21666 to correspond to the server's port. (You may have to launch Isaac a first time before having access to this file)
    5. Launch Isaac, choose your character, then start a run.
    6. Enter in the debug console by pressing ~ (QWERTY), ù (AZERTY) or ö (QWERTZ), depending on your keyboard layout.
    7. Type connect, space, then the server's IP.
    8. Enjoy !

    For more information, please refer to the Client Commands page.

    Server Setup, Step by Step

    1. Follow steps 1 and 2 of the client setup.
    2. Start Isaac a first time and close it.
    3. Extract the IsaacServer folder where you want, and put it on the host machine.
    4. Open config.json.
    5. Modify the seed to the seed you want each client to start with.
    6. If the port 21666 is already used, please choose another one, please indicate this to whoever will connect to your server, they will need it.
    7. Modify "maxClient" to the maximum number of clients the server will accept.
    8. You may need to do a port forwarding if you host your server yourself, please refer to your router's manual.
    9. Start the server.
    10. Enjoy.

    For more information, please refer to the Server Config page.
